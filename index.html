Hugging Face Agents Course Companion

LLM이 혼자 답하는 순간에서, 도구를 쓰는 에이전트로.

이 프로젝트는 Hugging Face Agent Course의 핵심 흐름을 한국어 인터랙티브 강의로 재구성합니다. 모델의 추론, 도구 호출, 관찰 루프가 하나의 문제 해결 시스템으로 이어지는 과정을 시각적으로 따라갑니다.

Project Purpose

이 사이트가 설명하는 한 가지 질문

LLM은 언어를 잘 다루지만, 최신 정보 조회나 계산, 파일 처리처럼 실제 세계와 맞닿은 작업은 외부 능력이 필요합니다. 강의는 이 간격을 스토리로 좁힙니다. 사용자의 목표가 들어오고, 모델이 계획을 세우고, 도구가 실행되고, 관찰 결과가 다시 추론으로 돌아오는 루프를 직접 확인하게 합니다.
